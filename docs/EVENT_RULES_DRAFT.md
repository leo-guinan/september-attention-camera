# September receipt game — draft rules

Goal: use audience-side cameras to capture evidence of memetic influence, energy, and growth.

## Basic idea

Participants install the September Attention Camera and browse public player surfaces. Each viewed profile/page can create a receipt. At the event, encoded camera rolls are revealed and scored.

## Suggested scoring dimensions

- Valid receipts: unique bounded observations with target, URL, timestamp, and visible stats.
- Attention: visible likes/reposts/replies/views/followers/subscribers where available.
- Diversity: receipts across multiple surfaces, not one refreshed profile forever.
- Trajectory: increasing visible stats over time for the same target/page.
- Trust: repeated high-quality interactions with consistent public evidence.
- Energy: high interaction density relative to audience size.

## Anti-gaming notes

- Duplicate identical captures should collapse.
- Synthetic deploy-smoke receipts do not count.
- Private or doxxing material is out of bounds.
- Hidden platform analytics only count if voluntarily exported by the account owner.
- Screenshots/media/OCR are future enrichment, not v1 truth.
